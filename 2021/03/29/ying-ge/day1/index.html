JAVA文件包含

  • 项目 ：JAVA中一个项目即为一个JAVA应用
  • 包： 为了内容的分类管理，在不同的包内有不同的类
    • 一般包名为域名的倒叙 ，几级则会在src文件夹中创建几个文件夹，例com.imooc 代表文件创建在src/com/imooc

Java特性

  • 跨平台
    • java针对每一个系统都提供了虚拟机(jvm)，只需要在不同的平台上安装上Jvm，在不同的系统上都可以运行Java代码
    • 首先由一个与平台无关的编译器生成.class文件，再在各个平台的环境上运行

Dos命令行执行java源代码：

  1. 将java源文件转换成字节码文件
    • 使用javac 当前文件名.java
  2. 执行代码
    • 使用java 类名

JRE与JDK

  • 使用JDK开发完成的java程序，交给JRE去运行

  • JRE包括Java虚拟机（JVM）和Java程序需要的核心类库

  • JVM：Java虚拟机，与计算机底层硬件交互

    • JVM执行.class文件

  • JRE：Java类库

  • JDK：Java开发工具包

  • JDK包含JRE包含JVM

字节码文件

  • JAVA运行之后在与包同名的out文件夹下生成一个字节码文件，(.class文件)，java虚拟机实际上运行.class

JVM虚拟机执行流程

  • 源码编辑后，通过javac编译生成字节码文件
  • 使用类加载器加载字节码文件.class
  • 执行引擎找到入口方法main()执行其中的字节码指令

数据类型

java对每一种数据都定义了具体的数据类型，在内存上分配了不同的内存空间

基本数据类型和引用数据类型区别

  1. 基本数据类型在定义的时候会在栈中开辟空间，而且这份空间所表达的值就是当前变量的默认值

    • 已经声明的变量会在栈中存在，改变值的过程实际上是覆盖过程

  2. 引用数据类型在定义的时候也会在栈中开辟内存，默认为null,引用数据类型如果赋值的话，要使用new关键字（创建对象的关键字），只要使用了new关键字，就会在堆内存中开辟空间

    //声明一个数据类型
Student student1 = new Student();

基本数据类型

  • 所有数据类型如果在类中定义的，要想在主函数中使用，需要将变量转换为静态static static int a = 5

  • 数值型

    • 整数类型(byte,short,int,long)
    • 浮点类型(float,double)
      • float在声明时后面要加f
        • float f = 10.5f

  • 字符型(char)

    • char c = '我';//只能有一个字符，多了报错

  • 布尔型(boolean)

    • true
    • false
    • 使用
      • 在class中定义要想在主函数中使用，需要声明是静态变量，
        • static boolean b = true

引用数据类型

  • 类(class)
  • 接口(interface)
  • 数组
  • 只要是类都为引用数据类型
  • 当引用数据类型使用New关键字时，则会在堆内存中开辟空间
String s = new String("sdhjdsah");

  • 引用数据类型也可以作为属性，此时m1只是一个空指针，要使其产生作用，需要new为其在堆内存中开辟空间，才能使用其属性。

     Method01 m1 ; //此处创建的m1实际上只是拥有Method01的属性，并没有拥有m1的值
  
//创建新的对象
       Method02 method02 = new Method02();
  
       method02.m1 = method01;  //获取method01的属性，交给m1

  • 如果是基本数据类型，就是简单的赋值操作；引用 数据类型的话，赋值的为地址。

    String str1 = "你好"; //在栈中开辟空间，在字符常量池中取值

    package case1.day1;

import java.util.Scanner;

public class day01_2 {
    public static void main(String[] args) {

        //创建第一个输入的数
        Scanner scanner = new Scanner(System.in);

        //在控制台输入数值的类型
        System.out.println("请输入第一个数:");
        double num1 = scanner.nextDouble();
        
		System.out.println("请输入第二个数:");
        double num2 = scanner.nextDouble();
        
		System.out.println("请输入第三个数:");
        double num3 = scanner.nextDouble();
        

        double numMax = 0;  //定义中间值
        double[] doubles = new double[3]; //创建一个数组

        doubles[0] = num1;  //为数组赋值
        doubles[1] = num2;
        doubles[2] = num3;

        for (int i = 0; i <doubles.length; i++) {  //遍历所有数据
            if(doubles[i] > numMax){  //与中间值比较，最大输出
                numMax = doubles[i];
            }

        }
        System.out.println("此时最大的值为："+numMax);

    }

}
