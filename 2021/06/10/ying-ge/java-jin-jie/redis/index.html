摘要

Redis 是nosql数据库，可以用于存放前端的数据，用作单点登录。

Redis

数据类型

  • Redis支持 5 种数据类型

    1. string(字符串)
    2. hash(哈希)
    3. list(列表)
    4. set(集合)
    5. zset (有序集合)

Redis keys命令

命令 描述
DEL key 用于在key存在时删除key
DUMP key 序列化给定key,并返回被序列化的值
EXISTS key 检查给定key是否存在
EXPIRE key seconds 为key设置过期时间
RENAME key newkey 修改key的名称
TYPE key 返回Key存储的值类型

Redis 字符串命令

命令 描述
SET key value 设置指定Key值
get Key 获取指定key的值
