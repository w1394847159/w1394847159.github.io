day7_HTML

发表于 2021-08-05 分类于 前端

摘要

本文介绍HTML语言，还记录了后端接收前端数据

HTML

  • 超文本标记语言

  • 页面由三部分组成：内容（结构），表现，行为

  • 内容是我们在页面中可以看到的数据，一般用html展示

  • html中每一个标签都属于一个dom对象

HTML与JAVA区别

  • Java文件需要先编译，再由java虚拟机跑起来
  • HTML文件不需要编译，直接由浏览器解析执行

HTML 文件书写规范

<html>
    表示整个Html页面开始
    <head>
        头信息
        <title>标题</title>
    </head>
</html>

HTML标签

html标签 内容 注意
普通文本框
文本域标签 可以输入多行文本
密码输入框
隐藏域标签， 客户端看不见，但真实存在
单选框 必须根据name分组，否则出现同时选中情况
多选框 根据name分组，否则出现同时选中情况
checked默认被选中状态
下拉列表框
表单提交按钮 需要在form中定义action= “地址”


水平线
每一个标签独自占一行
每一个标签不独自占一行，可与
嵌套

常用标签

多选框

  • 多选框标签，根据name分组，checkde为默认被选中状态

    <input type = "checkbox" name = hobby>

  • 多选框标签可以通过按钮事件执行全选和全不选功能

    //完成多选框全选的功能
            let qxElement = document.getElementById("qx");

            //绑定单击事件
            qxElement.onclick = function (){

                //获取所有多选框标签 通过name获取所有多选框
                let hobbyElement = document.getElementsByName("hobby")

                //遍历多选框
                for (let i = 0; i < hobbyElement.length; i++) {

                    //多选框的checked可以默认选中多选框，在script中可以直接使用dom对象的属性
                    hobbyElement[i].checked = true;

                }

字体标签

font 标签是字体标签，可用来修改文本字体，颜色，大小

color 属性修改颜色

face 属性修改字体

size 属性修改文本大小

<!--在网页上显示，我是字体标签，并修改字体为 宋体 颜色为红色-->

<font color="red" face="宋体">我是字体标签</font>

标题标签

  • h1 - h6都是标题标签

  • h1最大

  • h6最小

  • <h1 align= "center">标签

超链接

<a href = '绝对路径' >

  • target= 有两个属性值

    • taget="_self" 在自己标签内跳转

    • target="_blank" 创建一个新的标签页再跳转

列表标签

  • 有序列表

    <!--有序列表-->
<ol>
    <li>刘能</li>
    <li>赵四</li>
    <li>小沈阳</li>
    <li>宋小宝</li>
</ol>

  • 无序列表

    • <ul>是无序列表父标签,嵌套<li>

      <!--无序列表-->
<ul>
    <li>刘能</li>
    <li>赵四</li>
    <li>小沈阳</li>
    <li>宋小宝</li>
</ul>

img标签

  • img标签可以在html页面显示图片

  • src属性导入的是当前img地址

    • ./代表当前文件夹当前目录
    • ../代表当前文件夹上级目录
    <img src="../img/2.jpg" width="50%" height="50%">
    <img src="../img/2.jpg" width="200px" height="100px">

表格标签(重点)

  • <table> 是表格标签

    border 设置表格边框 border = "1"

    width 设置表格宽度

    height 设置表格高度

    align 设置表格相对于页面的对齐方式

    cellspacing 设置单元格间距

    tr 是行标签

    td 是表头标签

    th 表头（会自动加粗）

表单标签

  • 表单是一个包含单元素的区域

  • 表单元素： 允许用户在表单中（文本域，下拉列表，单选框，复选框）输入信息的元素

    <form>
  <input>
  <input>
</form>

  • 表单动作属性Action

    • 用户单击确认按钮时，表单内容会被传送到另一个文件。

标签语法

  • 标签不能交叉嵌套

  • 属性写在标签内，必须要用""括起来

    <font color = "blue">早安，尚硅谷</font>

  • 注释不能嵌套

后端接收数据

//文本，密码，单选都可以使用getParameter接收;
String name = req.getParameter("name");
        String age = req.getParameter("age");
        String sex = req.getParameter("sex");

//多选框传递值时，使用getParameterValues接收，返回值类型为数组
