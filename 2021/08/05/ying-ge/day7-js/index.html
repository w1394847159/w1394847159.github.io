摘要

JavaScript是前端的语言，可以为页面添加各种功能。

JavaScript

  • 主要是完成页面的数据验证，运行在客户单，需要运行浏览器来解析执行JavaScript代码。简称JS
  • js的注释和java是一样的，有多行注释和单行注释

Js代码写入地点

  1. 写在html的head下，在html文件中定义script标签，所有代码必须写在标签内

    <script type="text/javascript">

    //网页弹窗
    alert("这是我写的第一个程序！")

</script>

  2. 引入第三方js文件

    <script src="test"></script>

JS代码

网页弹窗

//网页弹窗
    alert("这是我写的第一个程序！");
        
//接收数据类型
alert(typeof object.name)

变量

  • 可是使用 var/let接收任意数值类型

    //声明一个变量
    var i =100;

//将结果显示在浏览器弹窗中
    alert(typeof i);   
         i = true;
    alert(typeof i);

JS数据类型

  • typeof 用于判断数据的数据类型

  • js共有 number,string,boolean,object,undefined,function,NaN七种数据类型

    var a = 10
var b = '2'
var c = true
var d = null
var e = undefined
var f = []
var g = {}
var h = function(){}
console.log(typeof a); // number
console.log(typeof b); // string
console.log(typeof c); // boolean
console.log(typeof d); // object
console.log(typeof e); // undefined
console.log(typeof f); // object
console.log(typeof g); // object
console.log(typeof h); // function
console.log(typeof a / b); // NaN

关键字NaN

  • Js的关键字，指不是数据类型
alert(i+j);  //字符串拼接

  //Number(i) 将数据转变成数值类型
  //alert(Number(i) + Number(j))  Nan js的关键字 指不是数值类型

Undefind

  • 表示未定义，当前变量没有赋值，在js中，所有变量都没有初始值，必须要定义

    var x;
 alert(x) //undefind 表示未定义，当前变量没有赋值 在js中，所有变量没有初始值

Null

  • Null实际上就是空值，可以照样赋值

    var x = null;
 alert(x) //null也是js中的关键字

逻辑运算

js中为假的数据

null false undefined 0

与运算（&&）

  • 布尔类型

    • 一个为假即为假

  • 不是布尔值

    • 有一个为假即为假
    • 全部为真，显示前者数据
    • 全部为假，显示前者数据

或运算 （||）

  • 正常的布尔类型
    • 有一个为真则为真
  • 不是布尔类型
    • 有一个为真即为真
    • 全部为真，取前者
    • 全部为假，取后者

取反（！）

  • 真即为假，假即为真

数组

  • js中的数组是不用规定长度的

  • 存放多少数据，长度就为多少

  • 可以存放任意数据类型

  • 如果直接定义到第n个,则未定义的数据都是undefined数据类型

    //在js中定义数组

    var array = [];

    array[0] = true;
    array[1] = false;
    array[2] = 100;
    array[3] = 2.5;

    //js中的数组是不用规定长度的
    for (var i = 0; i <array.length ; i++) {

        alert(array[i])

    }

 //js中另外一种遍历方式
    for (var i in array){
        alert(i)
    }

函数

  • 由于js是弱类型的语言，Js函数中的参数，是不需要指定数据类型的，直接写变量名称

     //声明一个函数 （）内体现出不需要指定数据类型
    function getFunction01(name,password){

        alert(name)
        alert(password)
    }
    //调用函数
getFunction01("jack","123456")

  • js是弱类型的语言，返回值不需要指定数据类型

    function getFunction02(num1,num2)
   {
       //强制转换为数值类型
       var sum = Number(num1) + Number(num2)
       return sum;
   }
  
   alert(getFunction02("100","200"))  //300

  • js中的函数是不支持重载的，定义两个函数名相同的函数，则后者会覆盖前者。

    //js不支持重载，函数名相同的函数后者会覆盖前者
    function getFunction03(){

        alert(arguments)
        alert("前者")
    }
    function getFunction03(){

        alert(arguments)
        alert("后者")
    }

getFunction03();  //返回后者

  • js函数定义中有隐藏属性 arguments 是数组对象

    • 没有定义变量也可以传参数进去，根据数组对象接收数据

       function getFunction04(){

        for (let i = 0; i < arguments.length; i++) {

            alert(arguments[i])
        }
    }

//调用函数
getFunction04("jack",12)

函数定义方式

  • 声明一个函数类型

    var fun = function (){

        alert("我是一个函数！！")
    }

    //调用函数
    fun()

  • 直接声明函数

    //js中函数的第二种定义方式
    function fun2(){
        alert("我是第二个函数")
    }
    fun2()

JS事件

  • 事件是可以被js侦测到的行为
事件 内容 作用
onload 加载完成事件 浏览器在解析Html最后一个标签之后触发的操作，页面加载完成之后，常用语js代码初始化操作
onclick 单击事件 单击某个按钮或某个模块时执行的事件
onblur 失去焦点事件 用于输入框失去焦点后验证输入内容是否合法
onchange 内容发生改变时间 用于下拉列表和输入框内容发生改变后操作
onsubmit 表单提交时间 用于表单提交前，验证所有表单项是否合法

注册（绑定）

静态注册

  • 直接根据Html标签内部属性绑定事件

动态注册（常用）

  • 根据js获取html的dom对象(html中，每个标签都属于一个dom对象)，再通过dom对象.事件名 = function(){ 函数体}

定义事件步骤

  • 获取事件对象 var btn = document.getElementById("btn")
  • 给事件对象绑定一个事件类型 var clickme = function(){}
  • 编写事件处理函数

    • 页面加载完成后执行的事件

DOM模型和document对象

  • DOM 全称 Document Object Model 文本对象模型

document对象

  • 创建的document对象都可以通过.属性的形式调用标签的属性 例如：.value获取value的值

  • docyment.getElementById(elementId) 通过标签Id属性查找标签dom对象 elementId是标签的id属性值

  • document.getElementsByName(elementName)

    • 通过标签的Name属性查找dom对象，elementId是标签的id属性值

  • document.getElementsByTagName(tagname) 通过标签名查找dom对象（即input,body等）

正则表达式

  • 只有定义正则字符串才能调用正则方法

判断是否包含中文字符

//获取输入的字符串
                var str = user.value;

                //创建字符串对象
                var reg = new RegExp("[\\u4E00-\\u9FFF]+","g");

                //判断是否是中文输入,如果是弹窗报错
                if (reg.test(str)){
                    alert("您输入的用户名包含中文，请重新输入")
                }

判断是否包含数字字母下划线

password.onblur =  function(){
                var str = password.value;

                //创建正则对象
                var reg = new RegExp("^[0-9a-zA-Z_]")

                //判断是否包含数字字母下划线，不包含则报错
                if (reg.test(str)){
                    alert("您输入的密码包含数字字母下划线")
                }else {
                    alert("您输入的密码包含其他字符")
                }
            }

jQuery

  • jQuery也就是 JavaScript 和查询（Query），它就是辅助 JavaScript 开发的 js 类库

  • 核心思想： 写的更少，做的更多

  • jQuery是为了方便Js开发封装的一个框架

  • 要想使用jquery框架，必须导入jquery依赖（jquery控件

    <!--引入jquery控件,使用相对路径-->
<script src="../jQuery/jquery-1.7.2.js">

  • 通过jquer调用的是方法，通过原生js调用的是dom属性

$ (函数)

  • jQuery中的 $ 是一个核心函数，可以完成jQuery的很多功能。$()即调用$这个函数。

  • 传入参数为[函数]时：表示页面接在完成之后，相当于window.onload = function(){}

  • jQuery文本就绪函数 传入参数为函数

    //jQuery文本就绪函数
$(function (){
    
})

//Js原生文本就绪函数
window.onload = function(){
    
}

  • 传入参数为[ html 字符串] 时：会为我们创建这个html标签对象

  • 传入参数为[ DOM对象] 时：把dom对象转换为jQuery对象

jQuery对象和dom对象

Dom对象

  • 通过getElementById()查询出来的标签对象是Dom对象
  • 不加$的都是Dom属性

jQuery对象

  • 通过jQuery提供的API创建的对象，是jQuery对象

  • 通过JQuery包装的Dom对象，也是JQuery对阿星

  • 通过JQuery提供的API查询到的对象，是JQuery对象

  • $() jQuery对象

Dom对象和jQuery对象互转

dom 对象转化为 jQuery 对象

  • 先有 DOM 对象 this

  • $( DOM 对象 ) 就可以转换成为 jQuery 对象 $(this)

jQuery 对象转为 dom 对象

  • 先有 jQuery 对象 $(this)

  • jQuery 对象[下标]取出相应的 DOM 对象 $(this[0])
