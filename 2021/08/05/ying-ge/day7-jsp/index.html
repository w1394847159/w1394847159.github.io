day7_JSP

发表于 2021-08-05 分类于 后端 ， java

摘要

本文简要介绍了JSP九大内置对象以及四大域对象，包含了EL表达式。

JSP

  • Java Server Pages 为了解决动态生成HTML文档的技术
  • 底层是JAVA
  • jsp的本质 实际上就是一个Servlet程序
  • JSP实际上是一个模板引擎，主要作用将程序渲染到浏览器上，

JSP头部指令

  • <%@ page contentType="text/html;charset=UTF-8" language="java" %>
  • contentType属性，表示jsp返回的数据类型是什么，可以用来解决响应时中文乱码
    • resp.setContentType("tex/html;charset=UTF-8")

JSP底层原理

<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<html>
<head>
    <title>测试</title>
</head>
<body>

        <h1>JSP的本质是一个Servlet</h1>

</body>
</html>
    
    

        //设置中文乱码，让浏览器支持html文本格式
        resp.setContentType("text/html;charset=UTF-8");

        //模拟Jsp的底层
        resp.getWriter().print("<html>\n");
        resp.getWriter().print("<head>\n");

        resp.getWriter().print("</hea>\n");

        resp.getWriter().print("</body>\n");
        resp.getWriter().print("</html>\n");

JSP脚本

  • 声明脚本的格式： <%! 声明 java 代码 %>

  • 表达式脚本的格式：<%=表达式%>

  • 代码脚本的格式： <% java 语句 %> 常用来写java代码

JSP九大内置对象

  1. request

  2. response

  3. pageContext

  4. exception

  5. application

  6. out输出

  7. page

  8. session

  9. servletConfig

JSP四大域对象

  • 从小到大的顺序：pageContext 、Request、Session 、application
域对象 作用 取值范围
pageContext JSP上下文文本对象 当前JSP
request 一次请求，多次请求无法访问
session 从浏览器开启到结束（关闭浏览器，则域对象失效）
application 实际上就是ServletContext对象 整个工程

EL表达式

  • ${域对象中的key}

  • EL表达式可以获取到域对象的值，还支持很多的逻辑运算，根据Key取值

  • 变量，对象，集合只有放到域对象中才可以被EI表达式获取

  • EL表达式调用属性会自动使用get方法

    //创建集合对象
HashMap hashMap = new HashMap();

//往集合传值
hashMap.put("abc","bbb")

    //随意表示一个key
application.setAttribute("hashMap",hashMap);

//<body>中
${pageContext1}<br>

  • EL表达式支持三目运算符，括号内使用key取值

    <%
application.setAttribute("num","10");
%>
    <body>
    ${num > 0 ? '测试成功' :'测试失败'}<br>
</body>

JSP排版

  • 由于JSP和Tomcat都未提供标签排版，所以需要导入第三方标签库(jstl标签库)进行排版，导入语句使用 代码脚本格式

    • 要想使用jstl，需要导入taglibs-standard-impl-1.2.1.jar包和taglibs-standard-spec-1.2.1.jar，并在Tomcat中也导入一次
    <!--
要想在jsp页面使用jstl标签库，必须在jsp页面使用taglib指示元素定义前置名称和uri参考
prefix 定义前置名称
uri 统一资源标识符 sun公司url参考
-->
<%@taglib prefix="c" uri = "http://java.sun.com/jsp/jstl/core" %>
